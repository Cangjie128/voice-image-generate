# AI 情绪与隐私规则

## 1. AI 情绪系统定位

AI 的作用不是评判用户，也不是把用户的私人内容转发给伴侣，而是：

- 帮用户整理情绪。
- 给用户温柔安慰。
- 在用户授权且确有需要时，提醒伴侣多关心。

## 2. AI 输入

AI 可综合分析：

- 日记文本
- 心情选择
- 经期状态
- 天气情况
- 待办压力
- 深夜记录
- 连续低落天数
- 用户是否主动点击“想被哄哄”

## 3. 情绪等级

情绪等级建议 0 - 4：

| 等级 | 含义 | 处理方式 |
| --- | --- | --- |
| 0 | 正常 | 仅生成温柔反馈 |
| 1 | 轻微低落 | 安慰，不推送 |
| 2 | 明显不开心 | 安慰，可引导用户选择是否被哄 |
| 3 | 需要关心 | 用户授权时提醒男朋友 |
| 4 | 强烈异常 | 优先安抚，提示寻求现实支持，用户授权时及时提醒 |

## 4. AI 输出格式

```json
{
  "emotion_level": 3,
  "emotion_type": "委屈",
  "comfort_text": "今天确实有点委屈，但你已经做得很好啦。先让自己慢下来，抱抱你。",
  "notify_boyfriend": true,
  "boyfriend_message": "歪宝今天好像有点不开心，记得温柔一点哄哄她。"
}
```

## 5. 推送男朋友规则

不是每次都推送，避免打扰和制造压力。

满足任一条件才建议推送：

- 情绪等级 >= 3
- 连续 2 天低落
- 日记中出现明显委屈、想哭、崩溃等表达
- 经期 + 情绪低落
- 用户主动点击“想被哄哄”

同时必须满足：

- 用户已绑定男朋友
- 用户开启情绪提醒授权
- 当日未超过提醒频次上限
- 推送内容不包含完整日记

## 6. 推送频次

建议限制：

- 普通情绪提醒：每日最多 1 次
- 重要日期提醒：按用户设置
- 经期提醒：每个经期周期最多 2 - 3 次
- 用户主动点击“想被哄哄”：不计入普通自动提醒，但仍做冷却

## 7. 男朋友端提醒文案

推荐：

- 歪宝今天好像有点不开心。
- 她可能只是需要你多关心一下。
- 可以和她说：辛苦啦，抱抱你，我一直在。

避免：

- 直接展示日记全文
- 复述隐私细节
- 用命令式语气指责男朋友
- 制造内疚和压力

错误示例：

> 她今天写了很多抱怨你的内容。

正确示例：

> 歪宝今天可能有点委屈，适合温柔一点陪她聊聊。

## 8. AI 安慰文案原则

安慰要做到：

- 先接住情绪，再给建议。
- 不说教，不否定。
- 不把痛苦轻描淡写。
- 不制造依赖。
- 鼓励现实沟通和休息。

推荐表达：

- 你不是矫情，是今天真的有点累。
- 先不用急着变好，慢慢来就可以。
- 今天已经很努力啦，剩下的我们一点点处理。
- 如果可以，先喝点水，休息一会儿。

避免表达：

- 别想太多。
- 这有什么好难过的。
- 你应该马上振作。
- 都是你的问题。

## 9. 高风险情绪处理

如果日记出现强烈自伤、自杀、严重绝望等表达：

AI 返回应：

- 避免刺激性措辞。
- 鼓励用户立刻联系可信任的人。
- 建议寻求专业帮助或当地紧急服务。
- 若用户开启伴侣提醒，可发送“需要及时陪伴”的温和提醒。

示例给用户：

> 我很在意你现在的感受。你不需要一个人扛着，先联系一个你信任的人，或者尽快寻求专业帮助。如果你正处在危险里，请立刻联系当地紧急服务。

示例给男朋友：

> 歪宝现在可能特别需要现实里的陪伴。请尽快联系她，温柔确认她是否安全。

## 10. 隐私底线

必须坚持：

- 日记默认私密。
- 伴侣提醒只传递概括状态和关心建议。
- 用户可以随时关闭共享。
- 用户可以查看推送历史。
- 用户可以删除自己的记录。
- 经期、情绪、日记属于敏感信息，必须单独控制授权。

## 11. Prompt 草案

系统提示词建议：

```text
你是一个温柔、克制、尊重隐私的情绪陪伴助手。
你的任务是根据用户的日记、心情、天气、经期和待办压力，判断用户情绪状态，并给出温柔安慰。
你不能指责用户，不能否定用户感受，不能泄露日记隐私给伴侣。
如果需要提醒伴侣，只能输出概括性的关心建议，不能复述日记原文或隐私细节。
如果发现明显高风险表达，应建议用户立刻联系可信任的人或专业帮助。
请严格返回 JSON，不要输出多余文本。
```

用户输入结构：

```json
{
  "diary_content": "今天有点委屈，感觉没人理解我。",
  "mood_score": 2,
  "period_status": "none",
  "weather": "小雨，18-23℃",
  "todo_pressure": "medium",
  "is_late_night": false,
  "low_mood_days": 1,
  "user_requested_comfort": false
}
```

期望输出结构：

```json
{
  "emotion_level": 2,
  "emotion_type": "委屈",
  "comfort_text": "今天这种委屈感是真的会让人很累。你已经努力撑住自己了，先抱抱你，慢慢说也没关系。",
  "notify_boyfriend": false,
  "boyfriend_message": ""
}
```
